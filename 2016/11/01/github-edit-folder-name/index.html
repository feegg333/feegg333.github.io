如何修改 GitHub 上的資料夾名稱

by Wei Liu (劉書瑋) — on 01 Nov 2016

Tag: GitHub

如同上一篇刪除 GitHub 上的資料夾，如果只是需要更改資料夾名稱，那就不用大費周章的刪除資料夾，又重新建立新的資料夾，打個指令就能夠輕鬆又快速的更改名稱嘍!

這次的目標就是要把 test02 這個資料夾名稱更改成為 test03。

之前先建立好了兩個資料夾，分別是 test01 與 test02。

首先開啟自己習慣的 指令工具，這邊是使用 Git Shell。

更改名稱指令，可用 git mv 「old name」 「new name」，如下方所示：

輸入這項指令後，你會看到他已經把該資料夾名稱從 test02 更改為 test03，接著別忘了打上 commit。

最後別忘了使用 push 指令，任務才算完成!

我們到 GitHub 上面看看， test02 資料夾名稱已經被更改成為 test03 。

非常簡短的指令，就輕鬆改好 GitHub 上的資料夾名稱嘍。
